I chose Nitro Type as my reference game. I think we can build a simplified version using only HTML, CSS, and JavaScript. Instead of copying the car racing idea exactly, we could use our own character/avatar. When the player types the code snippet correctly, the avatar moves forward. This keeps the game simple but still gives visual feedback and makes it feel more engaging than a normal typing test. We could also create custom avatars based on each team member to make the project feel more personalized and unique to our group.
